1 Overview

At Ciseco we like to make things simple and as such we have created a great range of power supplies. Called PowerPOD's these modular power supplies all have the same basic pin outs and a small footprint. Using this design in you project means you can drop in a diffrent PowerPOD as you power needs change. The simple inline pin out make them great for use in breadboard project.

 

2 Pinout and Dimesions

Each module has 4 pins, the first 3 are consistant across the range and the 4 is optional depending on the module.

  1. VIN - Power input your unregulated power input, voltage range will depends on the module
  2. GND - Ground
  3. VOUT - Power output, regulated output at the given voltage for the module
  4. Other - On some modules this can be used as the enable pin

All modules are 15mm wide. Most are 17mm long, the LiPo at 30mm,TPS62730 at 15mm AND TPS5402 at 70mm are the current exceptions.

 

3 Module Range

Not all of the modules listed below are available yet but we are working on them.

  • B011 - PowerPOD 1117 3V3
    • Regulator - 1117a LDO
    • INPUT - 16V Max
    • OUTPUT - 3V3 1A
    • Datasheet - AP1117

  • B012 - PowerPOD 1117 5V0
    • Regulator - 1117a LDO
    • INPUT - 16V Max
    • OUTPUT - 5V0 1A
    • Datasheet - AP1117

  • B013 - PowerPOD TC1015 3V3
    • Regulator - TC1015 LDO
    • INPUT - 6V Max
    • OUTPUT - 3V3 100mA
    • AUX - Enable Pin, This is pulled High (to VIN) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Datasheet - TC1015

  • B014 - PowerPOD TC1015 5V0
    • Regulator - TC1015 LDO
    • INPUT - 6V Max
    • OUTPUT - 5V0 100mA
    • AUX - Enable Pin, This is pulled High (to VIN) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Datasheet - TC1015

  • B015 - PowerPOD MIC5205YM5 Adjustable
    • Regulator - MIC5205YM5 Adjustable LDO
    • INPUT - 16V Max
    • OUTPUT - 3V3 or 5V0 (selectable via solder jumper) 150mA
    • AUX - Enable Pin, This is pulled High (to V???) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Other output voltages may be possible if you are willing to swap out the voltage divider resistors
    • Datasheet - MIC5205

  • B016 - PowerPOD NCP1402 3V3
    • Regulator - NCP1402 Set up switching regulator
    • INPUT - 1-3V
    • OUTPUT - 3V3 up-to 120mA with VIN of 1V5
    • AUX - Enable Pin, This is pulled High (to VOUT) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Datasheet - NCP1402

  • B017 - PowerPOD NCP1402 5V0
    • Regulator - NCP1402 Set up switching regulator
    • INPUT - 1-4V
    • OUTPUT - 5V0 up to 140mA with VIN of 3V0
    • AUX - Enable Pin, This is pulled High (to VOUT) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Datasheet - NCP1402

  • B018 - PowerPOD TPS62730 2V1
    • Regulator - TPS62730 Step down converter
    • INPUT - 4v2 Max
    • OUTPUT - 2v1 100mA
    • AUX - On/bypass, This is pulled High (to V???) via a 10K resistor to enable the regulator output by default but can be pulled LOW to enter Bypass mode which connects VIN to VOU via the internal bypass switch
    • STAT - Open drain status output with active low level.
    • Datasheet - TPS62730

  • B019 - PowerPOD TPS5402 3V3
    • Regulator - TPS5402 Step Down regulator
    • INPUT - 3V5 - 28V
    • OUTPUT - 3V3 1.7A
    • Datasheet - TPS5402

  • B020 - PowerPOD TPS5402 5V0
    • Regulator - TPS5402 Step Down regulator
    • INPUT - 28V Max
    • OUTPUT - 5V0 1.7A
    • Datasheet - TPS5402

  • B021 - PowerPOD LiPo
    • Regulator - TPS61200 Boost converter
    • INPUT(VIN) - 5V5 Max
    • OUTPUT - 3v3 300mA or 5V0 600mA selectable via solder jumper
    • AUX - Enable Pin, This is pulled High (to VIN) via a 10K resistor to enable the regulator output by default but can be pulled LOW to disable the regulator Output
    • Charge circuit - MCP73831 Single Cell LiPo charger
    • Charge Input - (+,GND) 6V0 MAX
    • Expansion PowerPOD - An extra PowerPOD can be chained onto the PowerPOD LiPo to regulate a high voltage Charge input down to the 6V0 need for the MCP73831
    • Datasheet - TPS61200
    • Datasheet - MCP73831
