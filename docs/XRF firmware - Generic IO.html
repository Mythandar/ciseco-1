XRF firmware - Generic IO

Details
Category: LLAP Devices - commands and instructions
Last Updated on Thursday, 28 May 2015 21:54

The suported commands for the generic IO firmware are:

Generic commands

  • APVER - LLAP version
  • DEVTYPE - Device type
  • DEVNAME - Device name
  • HELLO - Hello (PING)
  • SER - Serial number
  • FVER - Firmware version
  • CHDEVID - Change device ID
  • PANID - Change PANID
  • REBOOT - Restart the device

 Device specific commands

  • OUTA1 - Turn on output A (there are OUTA to OUTD available)
  • OUTA0 - Turn off output A (there are OUTA to OUTD available)
  • INPA - Returns the state of digital input A (the are INPA to INPD available)
  • ANAA - Returns the ADC value of ANAA (there are ANAA to ANAD)
  • SLEEPnnnX One shot Sleep for period of nnn where X is  T=Miliseconds, S= Seconds, H= Hours, D= Days
  • PLSA1 - Pulse output A high (there are PLSA to PLSD available)
  • PLSB0 - Pulse output B low (there are PLSA to PLSD available)
  • PLENnnnnn - Set length of pulse in miliseconds (1000 = 1 second, max 65.535 secs)
