All XRF Firmware Updates

Details
Category: Radio basics (XRF, URF, ERF & SRF)
Last Updated on Thursday, 19 February 2015 15:06

 

VERY IMPORTANT: Before upgrading your radio make sure you know the risks and fully understand what is required. Often, failed upgrades result in the radio being sent back to us for what is assumed to be a "hardware" problem.

THE RISKS: If the upgrade fails or the wrong combination of Bootloader and Firmware are used, the radio could stop working.

SHOULD I UPDATE? You will not require every update, so only update if you are 100% certain that it is necessary. The radio code has remained essentially unchanged for a long time. If you are not experiencing any issues and have a working system, we STRONGLY recommend you do not update the firmware.

DO I NEED TO UPDATE THE BOOTLOADER? If your device has a version prior to Serial 0.56, USB 0.30 or LLAP 0.47 then yes, you do need to update the bootloader before updating your firmware. Versions up from Serial 0.56, USB 0.30, or LLAP 0.47 already have the correct bootloader and do not need updating.

 

For information on firmware update procerude please see this page: Firmware updating - HOW TO

 

Release notes can be found here, please read to find out what has changed and whether you really need to update the firmware.
